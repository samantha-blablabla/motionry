# Motionry - Progress Report

## Project Overview
**Motionry** là một micro-animation library cho UI/UX, giúp developers dễ dàng tìm và sử dụng các animations cho web applications.

## Completed Features

### ✅ High Priority (Completed)
1. **Responsive Mobile Layout** - Mobile-first approach với Tailwind breakpoints
2. **Mobile Sidebar (Hamburger Menu)** - Slide-in menu cho mobile devices
3. **Hero/Landing Page** - Separate routes: Landing (`/`) và Library (`/library`)

### ✅ Medium Priority (Completed)
4. **Dark/Light Preview Toggle** - User có thể preview animation trên dark/light background
5. **Color Customizer** - User có thể pick colors cho animations, sync với code/prompt export
6. **Modal UI Redesign** - 2-column layout: Preview bên trái, Tabs (Custom + Prompt/Code) bên phải

## Color Configs Added
All 10 animations now have color configurations:
- Jelly Bounce: Background, Text Color
- Shimmer Stroke: Glow Color
- Magnetic Hover: Background, Text Color
- Toast Slide: Background, Text, Accent/Icon
- Search Expand: Background, Border, Icon Color
- Progress Fill: Fill Color, Track Color
- Delayed Tooltip: Background, Text Color
- Card Swipe: Card Background, Border
- Text Pop: Text Color, Hover Color
- Name Tag Reveal: Tag Background, Tag Text

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript
